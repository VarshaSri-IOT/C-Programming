# 🔁 Loops in C Programming

This module contains programs written while learning looping statements in C.

---

# 📚 Topics Covered

## 1️⃣ Basic Programs (`01.Basic-programs`)

Programs to understand loop syntax and simple iterations.

### Programs:

* Print numbers from 1 to 10
* Print even numbers
* Print odd numbers
* Multiplication tables

### Concepts Used:

* `for` loop
* `while` loop
* `do-while` loop
* Increment and decrement operators

---

## 2️⃣ Digit Programs (`02.Digit-programs`)

Programs that process individual digits of a number using `%` and `/`.

### Programs:

* Count digits
* Reverse a number
* Sum of digits
* Palindrome number
* Armstrong number
* Strong number

### Concepts Used:

* Extracting digits
* Loop repetition
* Temporary variables
* Mathematical logic

---

## 3️⃣ Number Programs (`03.Number-programs`)

Programs involving mathematical properties of numbers.

### Programs:

* Factorial
* Fibonacci Series
* Prime Number
* Sum of Natural Numbers

### Concepts Used:

* Nested loops
* Mathematical formulas
* Condition checking
* Iterative calculations

---

## 4️⃣ Mini Projects (`Mini-projects`)

### Number Analysis System

A single program that checks:

✔ Even/Odd
✔ Positive/Negative
✔ Prime Number
✔ Reverse Number
✔ Sum of Digits

### ATM Management System

Simple menu-driven project using loops and conditions.

Concepts Used:

* Menu-driven programming
* Infinite loops
* `switch` statements
* Nested conditions

---

## 5️⃣ Pattern Programs (`Pattern-programs`)

Programs for printing patterns using nested loops.

Examples:

* Star patterns
* Number patterns
* Pyramid patterns
* Inverted patterns

Concepts Used:

* Nested `for` loops
* Row and column logic
* Pattern thinking
