# 📚 Arrays in C Programming

This folder contains programs related to **Arrays in C**, one of the most important concepts in programming and a foundation for Data Structures and Algorithms.

* Input and Output of Arrays
* Traversing Arrays
* Finding Sum of Elements
* Finding Average of Elements

Programs:

* `input-array.c`
* `sum-array.c`
* `Average-element.c`

* Largest Element in an Array
* Smallest Element in an Array

Programs:

* `largest-element.c`
* `smallest-element.c`

* Deleting Elements from Sorted Arrays

Programs:

* `inserting-elements.c`
* `element-deletion-sorted-array.c`
* `element-deletion-unsorted-array.c`
