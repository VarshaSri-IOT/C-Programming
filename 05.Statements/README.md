# 05. Statements

This module introduces **control statements** in C, which allow a program to make decisions and control the flow of execution.

The programs in this section cover conditional statements, nested conditions, and switch-case implementations through practical examples.

---

## 📚 Topics Covered

- if statement
- if-else statement
- else-if ladder
- nested if
- switch statement

---

## 📂 Programs

| Program | Description |
|----------|-------------|
| `if-else-statement-voting-eligibility.c` | Checks whether a person is eligible to vote. |
| `if-statement-leap_year.c` | Determines whether a given year is a leap year. |
| `else-if-ladder-tax-calculator.c` | Calculates tax based on income slabs using an else-if ladder. |
| `nested-if-sorting-order.c` | Sorts three numbers in ascending and descending order using nested if statements. |
| `switch-statement.c` | Prints the month name corresponding to the entered month number. |
| `Mini-project-Guessing-number.c` | A simple number guessing game demonstrating decision-making statements. |

---

## 🎯 Learning Objectives

After completing this module, I learned how to:

- Write decision-making programs using `if`.
- Choose between multiple conditions using `else-if`.
- Implement nested conditions.
- Use logical operators (`&&`, `||`, `!`) effectively.
- Replace long conditional chains with `switch`.
- Build small interactive console programs.
